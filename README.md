# Ask Finance - AI Finance Assistant Architecture

## Executive Summary

Ask Finance is an enterprise-grade AI-powered financial assistant designed to serve as a virtual financial business partner for organizations with multiple business units and regions. The system enables natural language queries about financial data, provides role-based access control, and generates insights in multiple formats (text, Excel, PowerPoint).

### Key Capabilities

- **Natural Language Understanding**: Comprehends financial terminology (P&L, Opex, ROI, variance, cash flow, EBIT, etc.)
- **Multi-Source Data Integration**: Consolidates data from SAP, HFM, data lakes, and other enterprise systems
- **Role-Based Access Control**: Enforces fine-grained permissions at row, column, and dataset levels
- **Multi-Format Output**: Generates responses in text, visualizations, Excel, and PowerPoint
- **Explainable AI**: Provides citations and reasoning for all answers
- **Multi-Agent Architecture**: Scalable design with specialized agents for different financial domains
- **Real-Time & Historical Analytics**: Combines batch and streaming data processing

### 1.4 Chunking & Embedding Strategies

**Chunking Strategy Selection Matrix:**

| Data Type | Strategy | Chunk Size | Overlap | Rationale |
|-----------|----------|------------|---------|-----------|
| Financial Reports | Semantic Splitting | Variable (100-500 tokens) | 50 tokens | Preserves financial statement structure |
| Policy Documents | Recursive Character | 500 tokens | 100 tokens | Maintains policy context |
| Transaction Data | Metadata-Enhanced | By transaction group | N/A | Grouped by entity/period |
| Metric Definitions | Document-Specific | By metric | N/A | Each metric is atomic |

**Embedding Strategy - Hybrid Approach:**

1. **Dense Embeddings**: Use `text-embedding-3-large` or `text-embedding-3-small` for primary semantic search
2. **Sparse Embeddings**: Use BM25 for keyword matching on financial terms
3. **Metadata Filtering**: Pre-filter by user permissions, time period, business unit before vector search
4. **Parent-Child Relationships**: Store small chunks with references to parent documents for context expansion

**Advanced Indexing Techniques:**

```python
# Pseudo-code for multi-strategy embedding
class FinancialDocumentEmbedder:
    def embed_document(self, document):
        # Strategy 1: Chunk main content
        chunks = semantic_split(document.content)

        # Strategy 2: Generate summary embedding
        summary = generate_summary(document.content)
        summary_embedding = embed(summary)

        # Strategy 3: Generate hypothetical questions
        questions = generate_hypothetical_questions(document.content)
        question_embeddings = [embed(q) for q in questions]

        # Strategy 4: Extract and embed key metrics
        metrics = extract_financial_metrics(document.content)
        metric_embeddings = [embed(f"{m.name}: {m.definition}") for m in metrics]

        return {
            'chunk_embeddings': [embed(c) for c in chunks],
            'summary_embedding': summary_embedding,
            'question_embeddings': question_embeddings,
            'metric_embeddings': metric_embeddings,
            'metadata': document.metadata
        }
```

**Purpose**: Performs advanced document retrieval with hybrid search and reranking.

**Retrieval Strategy**:

```mermaid

    REWRITE --> PARALLEL{Parallel Search}

    PARALLEL --> VECTOR[Vector Search<br/>Dense Embeddings]
    PARALLEL --> KEYWORD[Keyword Search<br/>BM25]
    PARALLEL --> METADATA[Metadata Filter<br/>RBAC Pre-filter]

    MERGE --> RERANK[Reranking<br/>Cross-Encoder Model]
    RERANK --> FINAL[Top N Results<br/>With Scores & Citations]
```

**Hybrid Search Implementation**:

```python
class HybridRetriever:
    def retrieve(self, query: str, user_context: dict, top_k: int = 10):
        # Step 1: Query rewriting
        expanded_query = self.expand_financial_terms(query)

        # Step 2: Metadata pre-filtering (RBAC)
        metadata_filter = self.build_rbac_filter(user_context)

        # Step 3: Parallel search
        vector_results = self.vector_search(
            query=expanded_query,
            top_k=top_k * 2,
            filter=metadata_filter
        )

        keyword_results = self.bm25_search(
            query=expanded_query,
            top_k=top_k * 2,
            filter=metadata_filter
        )

        # Step 4: Reciprocal Rank Fusion
        merged_results = self.rrf_merge(
            [vector_results, keyword_results],
            weights=[0.7, 0.3]
        )

        # Step 5: Reranking with cross-encoder
        reranked_results = self.rerank(
            query=query,
            documents=merged_results[:top_k * 2],
            top_k=top_k
        )

        return reranked_results
```

### 2.4 Model Context Protocol (MCP) Integration

**Why MCP?**
- Decouples agent logic from tool implementations
- Enables tool reusability across different AI systems
- Standardizes tool interfaces for easier maintenance
- Facilitates sharing tools with other teams

**MCP Architecture**:


```python
class MemoryExtractor:
    def extract_preferences(self, conversation_history: list) -> dict:
        """Extract user preferences from conversation history"""
        preferences = {}

        # Detect preferred output formats
        format_counts = self.count_format_requests(conversation_history)
        if format_counts:
            preferences['preferred_format'] = max(format_counts, key=format_counts.get)

        # Detect frequently queried business units
        bu_mentions = self.extract_business_unit_mentions(conversation_history)
        if bu_mentions:
            preferences['primary_business_units'] = bu_mentions[:3]

        # Detect preferred level of detail
        detail_level = self.infer_detail_preference(conversation_history)
        preferences['detail_level'] = detail_level

        # Detect time periods of interest
        time_periods = self.extract_time_period_patterns(conversation_history)
        preferences['typical_time_periods'] = time_periods

        return preferences
```

### 3.3 Data Masking Strategies

```python
class DataMasker:
    """Apply column-level security based on user role"""

    SENSITIVE_COLUMNS = {
        'employee_salary': ['salary', 'bonus', 'total_compensation'],
        'customer_pii': ['customer_name', 'email', 'phone'],
        'strategic_projects': ['project_name', 'investment_amount', 'roi']
    }

    def mask_data(self, dataframe: pd.DataFrame, user_role: str) -> pd.DataFrame:
        mask_level = self.get_mask_level(user_role)

        if mask_level == 'unmasked':
            return dataframe

        if mask_level == 'partial_mask':
            # Show aggregated data only
            for col in self.SENSITIVE_COLUMNS['employee_salary']:
                if col in dataframe.columns:
                    dataframe[col] = dataframe[col].apply(self.partial_mask)

        if mask_level == 'fully_masked':
            # Replace with masked values
            for category, cols in self.SENSITIVE_COLUMNS.items():
                for col in cols:
                    if col in dataframe.columns:
                        dataframe[col] = '***MASKED***'

        return dataframe

    def partial_mask(self, value):
        """Show range instead of exact value"""
        if pd.isna(value):
            return value

        # Round to nearest range
        if value < 50000:
            return "< $50K"
        elif value < 100000:
            return "$50K - $100K"
        elif value < 200000:
            return "$100K - $200K"
        else:
            return "> $200K"
```

```python
# agents/supervisor_agent.py
from langgraph.graph import StateGraph, END
from langgraph.prebuilt import ToolExecutor
from typing import TypedDict, Annotated, Sequence
import operator

class AgentState(TypedDict):
    query: str
    user_context: dict
    messages: Annotated[Sequence[dict], operator.add]
    agent_outcome: str
    intermediate_steps: list
    final_response: dict

def create_supervisor_agent():
    """Create the supervisor agent with LangGraph"""

    workflow = StateGraph(AgentState)

    # Define nodes
    workflow.add_node("classify_intent", classify_intent_node)
    workflow.add_node("route_agent", route_agent_node)
    workflow.add_node("qa_agent", qa_agent_node)
    workflow.add_node("retrieval_agent", retrieval_agent_node)
    workflow.add_node("analytics_agent", analytics_agent_node)
    workflow.add_node("report_agent", report_agent_node)
    workflow.add_node("synthesize_response", synthesize_response_node)

    # Define edges
    workflow.set_entry_point("classify_intent")
    workflow.add_edge("classify_intent", "route_agent")

    # Conditional routing based on intent
    workflow.add_conditional_edges(
        "route_agent",
        route_agent_decision,
        {
            "qa": "qa_agent",
            "retrieval": "retrieval_agent",
            "analytics": "analytics_agent",
            "report": "report_agent"
        }
    )

    # All agents flow to synthesis
    workflow.add_edge("qa_agent", "synthesize_response")
    workflow.add_edge("retrieval_agent", "synthesize_response")
    workflow.add_edge("analytics_agent", "synthesize_response")
    workflow.add_edge("report_agent", "synthesize_response")

    workflow.add_edge("synthesize_response", END)

    return workflow.compile()

def classify_intent_node(state: AgentState) -> AgentState:
    """Classify user intent"""
    query = state["query"]

    # Use LLM to classify intent
    intent = intent_classifier.classify(query)

    state["messages"].append({
        "role": "system",
        "content": f"Classified intent: {intent}"
    })

    return state

def route_agent_decision(state: AgentState) -> str:
    """Decide which agent to route to"""
    last_message = state["messages"][-1]
    intent = last_message["content"]

    if "simple_qa" in intent:
        return "qa"
    elif "document_search" in intent:
        return "retrieval"
    elif "analysis" in intent or "calculation" in intent:
        return "analytics"
    elif "report" in intent or "export" in intent:
        return "report"
    else:
        return "qa"  # Default to Q&A

# Main execution
supervisor = create_supervisor_agent()

def process_query(query: str, user_context: dict) -> dict:
    """Main entry point for query processing"""
    initial_state = {
        "query": query,
        "user_context": user_context,
        "messages": [],
        "intermediate_steps": [],
        "agent_outcome": "",
        "final_response": {}
    }

    result = supervisor.invoke(initial_state)
    return result["final_response"]
```

### 9.2 RBAC-Aware SQL Tool

```python
# tools/sql_query_tool.py
from typing import Dict, Any
import sqlalchemy as sa
from sqlalchemy.orm import Session

class RBACQueryTool:
    """SQL query tool with RBAC enforcement"""

    def __init__(self, engine: sa.Engine, rbac_engine: RBACEngine):
        self.engine = engine
        self.rbac_engine = rbac_engine

    def execute_query(
        self,
        query: str,
        user_context: Dict[str, Any],
        parameters: Dict[str, Any] = None
    ) -> Dict[str, Any]:
        """Execute SQL query with RBAC filtering"""

        # Get user permissions
        permissions = self.rbac_engine.get_permissions(user_context["user_id"])

        # Validate query doesn't access forbidden tables
        self._validate_query_access(query, permissions)

        with Session(self.engine) as session:
            # Set user context for RLS
            session.execute(
                sa.text(f"SET app.user_id = {user_context['user_id']}")
            )
            session.execute(
                sa.text(f"SET app.user_role = '{permissions['role']}'")
            )

            # Apply additional WHERE clause filters
            filtered_query = self._apply_rbac_filters(query, permissions)

            # Execute query
            result = session.execute(
                sa.text(filtered_query),
                parameters or {}
            )

            # Convert to dict
            columns = result.keys()
            rows = [dict(zip(columns, row)) for row in result.fetchall()]

            # Apply column-level security
            masked_rows = self._mask_sensitive_columns(rows, permissions)

            return {
                "status": "success",
                "data": masked_rows,
                "row_count": len(masked_rows),
                "columns": columns
            }

    def _validate_query_access(self, query: str, permissions: Dict):
        """Validate user can access tables in query"""
        # Parse SQL to extract table names
        tables = self._extract_tables(query)

        allowed_tables = permissions.get("allowed_tables", [])

        for table in tables:
            if table not in allowed_tables and "*" not in allowed_tables:
                raise PermissionError(
                    f"User does not have access to table: {table}"
                )

    def _apply_rbac_filters(self, query: str, permissions: Dict) -> str:
        """Add WHERE clause filters based on user permissions"""
        filters = []

        # Business unit filter
        if permissions.get("business_units") != ["*"]:
            bu_list = "','".join(permissions["business_units"])
            filters.append(f"business_unit IN ('{bu_list}')")

        # Region filter
        if permissions.get("regions") != ["*"]:
            region_list = "','".join(permissions["regions"])
            filters.append(f"region IN ('{region_list}')")

        # Time range filter
        if time_range := permissions.get("time_range"):
            filters.append(f"fiscal_period >= '{time_range['start']}'")
            filters.append(f"fiscal_period <= '{time_range['end']}'")

        if not filters:
            return query

        # Inject filters into WHERE clause
        where_clause = " AND ".join(filters)

        if "WHERE" in query.upper():
            return query.replace("WHERE", f"WHERE {where_clause} AND")
        else:
            return f"{query} WHERE {where_clause}"

    def _mask_sensitive_columns(
        self,
        rows: list,
        permissions: Dict
    ) -> list:
        """Apply column-level security masking"""
        mask_level = permissions.get("sensitive_data", "masked")

        if mask_level == "unmasked":
            return rows

        sensitive_columns = ["salary", "bonus", "ssn", "employee_id"]

        for row in rows:
            for col in sensitive_columns:
                if col in row:
                    if mask_level == "fully_masked":
                        row[col] = "***MASKED***"
                    elif mask_level == "partial_mask":
                        row[col] = self._partial_mask(row[col])

        return rows
```

### 9.3 Hybrid Retrieval Implementation

```python
# tools/hybrid_retrieval_tool.py
from typing import List, Dict
import numpy as np
from rank_bm25 import BM25Okapi
from sentence_transformers import CrossEncoder

class HybridRetrievalTool:
    """Hybrid retrieval with vector search and BM25"""

    def __init__(
        self,
        vector_store,
        embedding_model,
        reranker_model="cross-encoder/ms-marco-MiniLM-L-12-v2"
    ):
        self.vector_store = vector_store
        self.embedding_model = embedding_model
        self.reranker = CrossEncoder(reranker_model)

    def retrieve(
        self,
        query: str,
        user_context: Dict,
        top_k: int = 10,
        alpha: float = 0.7  # Weight for vector search vs BM25
    ) -> List[Dict]:
        """Perform hybrid retrieval with reranking"""

        # Step 1: Expand query with financial synonyms
        expanded_query = self._expand_financial_terms(query)

        # Step 2: Build RBAC metadata filter
        metadata_filter = self._build_rbac_filter(user_context)

        # Step 3: Vector search
        query_embedding = self.embedding_model.encode(expanded_query)
        vector_results = self.vector_store.similarity_search_with_score(
            query_embedding,
            k=top_k * 2,
            filter=metadata_filter
        )

        # Step 4: BM25 search
        # (Assume we have pre-indexed documents)
        bm25_results = self._bm25_search(
            expanded_query,
            metadata_filter,
            top_k=top_k * 2
        )

        # Step 5: Reciprocal Rank Fusion
        merged_results = self._reciprocal_rank_fusion(
            vector_results,
            bm25_results,
            alpha=alpha
        )

        # Step 6: Rerank with cross-encoder
        reranked_results = self._rerank(
            query,
            merged_results[:top_k * 2],
            top_k=top_k
        )

        return reranked_results

    def _expand_financial_terms(self, query: str) -> str:
        """Expand financial abbreviations and synonyms"""
        expansions = {
            "P&L": "Profit and Loss Statement",
            "Opex": "Operating Expenses",
            "Capex": "Capital Expenditures",
            "ROI": "Return on Investment",
            "EBIT": "Earnings Before Interest and Tax",
            "EBITDA": "Earnings Before Interest Tax Depreciation and Amortization",
            "YoY": "Year over Year",
            "QoQ": "Quarter over Quarter"
        }

        expanded = query
        for abbr, full in expansions.items():
            if abbr in query:
                expanded += f" {full}"

        return expanded

    def _build_rbac_filter(self, user_context: Dict) -> Dict:
        """Build metadata filter based on user permissions"""
        filters = {}

        if user_context.get("business_units") != ["*"]:
            filters["business_unit"] = {"$in": user_context["business_units"]}

        if user_context.get("regions") != ["*"]:
            filters["region"] = {"$in": user_context["regions"]}

        return filters

    def _reciprocal_rank_fusion(
        self,
        results_list1: List[tuple],
        results_list2: List[tuple],
        alpha: float = 0.7,
        k: int = 60
    ) -> List[Dict]:
        """Merge two ranked lists using RRF"""

        # Calculate RRF scores
        rrf_scores = {}

        for rank, (doc, score) in enumerate(results_list1):
            doc_id = doc.metadata["id"]
            rrf_scores[doc_id] = rrf_scores.get(doc_id, 0) + alpha / (k + rank + 1)

        for rank, (doc, score) in enumerate(results_list2):
            doc_id = doc.metadata["id"]
            rrf_scores[doc_id] = rrf_scores.get(doc_id, 0) + (1 - alpha) / (k + rank + 1)

        # Sort by RRF score
        sorted_docs = sorted(
            rrf_scores.items(),
            key=lambda x: x[1],
            reverse=True
        )

        # Retrieve full documents
        merged_results = []
        for doc_id, score in sorted_docs:
            # Find doc in either result list
            doc = self._find_doc_by_id(doc_id, results_list1, results_list2)
            if doc:
                merged_results.append({
                    "document": doc,
                    "rrf_score": score
                })

        return merged_results

    def _rerank(
        self,
        query: str,
        documents: List[Dict],
        top_k: int
    ) -> List[Dict]:
        """Rerank documents using cross-encoder"""

        # Prepare pairs for reranking
        pairs = [[query, doc["document"].page_content] for doc in documents]

        # Get reranking scores
        scores = self.reranker.predict(pairs)

        # Combine with RRF scores (weighted average)
        for i, doc in enumerate(documents):
            doc["rerank_score"] = scores[i]
            doc["final_score"] = 0.5 * doc["rrf_score"] + 0.5 * scores[i]

        # Sort by final score
        reranked = sorted(
            documents,
            key=lambda x: x["final_score"],
            reverse=True
        )

        return reranked[:top_k]
```
